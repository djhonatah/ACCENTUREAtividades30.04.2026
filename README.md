# 🚀 Academia Accenture - Atividades Java + IA

Este repositório contém as atividades práticas e projetos desenvolvidos durante a **Academia Accenture JAVA + IA**. 

O principal objetivo é aplicar e consolidar conhecimentos em desenvolvimento backend e inteligência artificial, seguindo rigorosamente as boas práticas de engenharia de software para garantir organização, legibilidade e escalabilidade do código.
